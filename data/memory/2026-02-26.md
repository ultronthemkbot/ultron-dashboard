### SSML Failure (Lesson Learned)
- Tried passing raw SSML XML to `edge_tts.Communicate()` — voice read xmlns URLs literally ("http www.w3.org...")
- Python API does NOT auto-parse SSML; CLI `--ssml` flag also not available in installed version
- Fix: plain text only + prosody params (rate, pitch) via Python API
- Pitch format in edge_tts: must be `+15Hz` NOT `+10%` (Hz only!)
